Web gis

## Génération automatique des icônes (Android & iOS)

Un script Node est fourni pour générer les PNG depuis les SVG (`images/icon-192.svg` et `images/icon-512.svg`).

Commandes :

1. Installer les dépendances :

   npm install

2. Générer les icônes :

   npm run generate-icons

Le script crée : `images/icon-192.png`, `images/icon-512.png`, `images/apple-touch-icon.png`.
